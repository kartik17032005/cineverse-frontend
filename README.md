# 🎬 CineVerse

CineVerse is an Android movie discovery application built using Kotlin and modern Android development tools.  
The app allows users to explore movies, view trailers, manage watchlists, and get AI-powered recommendations.

---

## ✨ Features

- 🔍 Search movies
- 🎥 Watch trailers
- ❤️ Add movies to watchlist
- 🤖 AI-powered recommendations
- 👤 User authentication
- 🌙 Modern UI

---

## 🛠 Tech Stack

### Frontend
- Kotlin
- Jetpack Compose / XML
- MVVM Architecture
- Retrofit
- Coroutines
- Room Database

### Backend
- Spring Boot
- MySQL
- REST APIs
